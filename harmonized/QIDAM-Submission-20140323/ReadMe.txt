This package contains materials for the HL7 Health Quality Improvement, Domain Analysis Model Release 1 for the May 2014 ballot.

The main specification document is HL7_DAM_HQUAL_R1_I1_2014___.pdf

In addition the following supplemental files are included

* QDM-vMR-cross-map.xlsx - a file showing mappings among QIDAM, QDM, and VMR specifications
* QIDAM.eap - a UML class model of QIDAM. A viewer for Microsoft Windows computers is available at http://www.sparxsystems.com/bin/EALite.exe
* QIDAM-htmldocs.zip - Documentation generated from the UML model. This document shows all class and interface attributes on one page.
